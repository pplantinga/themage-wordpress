Theme for DREDF's mobile pages.

== Description ==

DREDF created, manages, and uses this theme to display all mobile content.

== Installation ==

1. Upload `dredf-mobile` to the `/wp-content/themes/` directory
1. Activate the theme through the 'Theme' menu in WordPress
